Usage:
  dep-tree [flags]
  dep-tree [command]

Available Commands:
  check       Checks the that the current project matches the dependency rules defined in the configuration
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  render      [default] Render the dependency tree starting from the provided entrypoint

Flags:
  -c, --config string   path to dep-tree's config file
  -h, --help            help for dep-tree
  -v, --version         version for dep-tree

Use "dep-tree [command] --help" for more information about a command.
